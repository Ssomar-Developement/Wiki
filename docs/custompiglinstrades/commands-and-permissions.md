## Commands

#### Create a new Trade

* Command: ****/cpt create \{id\}****
* Permission: `cpt.cmd.create`

#### Open the editor / menu

* Command: ****/cpt editor**** or ****/cpt show****
* Permission: `cpt.cmd.editor` or `cpt.cmd.show`

#### Reload the plugin

* Command: ****/cpt reload****
* Permission: `cpt.cmd.reload`

#### Delete a Trade

* Command: ****/cpt delete \{id\}****
* Permission: `cpt.cmd.create`
